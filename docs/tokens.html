Content Tokens

These tokens standardize repeatable UI text, labels, and phrasing across the application, ensuring localization-readiness and speed in delivery.

Token Name Example Output Usage
content.cta.submit Submit Payment Used on primary action buttons
content.status.success Payment submitted successfully Used in success banners
content.tooltip.vendor_name This is the legal name used for tax reporting Tooltip for vendor field
format.currency.standard $#,##0.00 Standard format for all currency display
